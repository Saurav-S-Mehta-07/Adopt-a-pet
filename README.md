# Adopt-a-Pet

Adopt-a-Pet is a **frontend-only pet adoption website UI** created using **HTML, CSS, and JavaScript**. The project focuses on designing a clean, responsive, and user-friendly interface for a pet adoption platform.

This project does not include backend functionality or real pet listings. It is intended purely as a **UI/UX and frontend design prototype**.
